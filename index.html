Welcome

This is a companion site for the Web Design for Everybody course. The four main topics we will cover are HTML5, CSS3, JavaScript, and Responsive Design. The focus is on best practices and accessibility.

The best way to learn web design is with hands-on learning. While this site looks great, you can see how it was built step by step — starting with just HTML, adding CSS for layout, and then JavaScript for interactivity.

Whether you are a beginner or have some experience, this course will help you build modern, accessible web pages that look great on all devices.
